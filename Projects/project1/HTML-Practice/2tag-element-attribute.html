Tags-Element-Attributes

Tag

Tag are those which are come in between < AngularBracket>

90% tags are come with start tag and End Tag. for eg. h1, p, html. body, head etc.

10% tags are self Closing Tag or Void Tag or Empty Tag. For eg. br, hr, img, meta etc.

Element

It is an content between opening and closing in simple language.

Attribute

It provides some additional Information Through which behaviour of Element Changes. Attribute always written in Start Tag.
